Для коректной работы скрипта необходимо вставить СВОЙ токен дискорда, и скачть библиотеку, ниже инструкция.

Библиотека:
pip install discord.py-self (прописать в консоль)

ТОКЕН:
Пошаговая инструкция:
Зайди в Discord через браузер (Chrome, Edge, Opera и т.д.). Войди в свой аккаунт.
Нажми F12 или комбинацию Ctrl + Shift + I (на Mac: Cmd + Option + I), чтобы открыть панель разработчика.
В верхней части открывшейся панели найди вкладку Network (Сеть). Если её не видно, нажми на стрелочки >>.
Теперь отправь любое сообщение любому другу в ЛС или в какой-нибудь канал.
В списке слева (в колонке Name) появится много строк. Тебе нужно найти ту, которая называется science или messages.
Нажми на эту строку и справа выбери вкладку Headers (Заголовки).
Прокрути вниз до раздела Request Headers, пока не найдешь строку authorization.
Напротив неё будет длинная комбинация букв и цифр. Это и есть твой токен.

Кавычки не убирай, просто замени текст "ВАШ ТОКЕН" на ключ, должно быть примерно так
TOKEN = "" #Read readme_blyat

Запускай батник с правами админа и после добавления библиотеки и токена!
